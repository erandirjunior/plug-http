# Working with get

## Initialize
```php
use \PlugHttp\Globals\Env;

$env = new Env();
```

## Methods

>  Getting all values
```php
$env->all();
```

> Getting specific value
```php
$env->get($key);
```

> Getting all values except
```php
$env->except([$keyOne, $keyTwo]);
```

> Getting only values
```php
$env->only([$keyOne, $keyTwo]);
```

> Add value
```php
$env->add($key, $value);
```

> Remove value
```php
$env->remove($key);
```

> Check if has value
```php
$env->has($key);
```
