# Working with get

## Initialize
```php
use \PlugHttp\Globals\Get;

$get = new Get();
```

## Methods

>  Getting all values
```php
$get->all();
```

> Getting specific value
```php
$get->get($key);
```

> Getting all values except
```php
$get->except([$keyOne, $keyTwo]);
```

> Getting only values
```php
$get->only([$keyOne, $keyTwo]);
```

> Add value
```php
$get->add($key, $value);
```

> Remove value
```php
$get->remove($key);
```

> Check if has value
```php
$get->has($key);
```
